<h4 align="center">
Allows you to run Phaser 3 game (including Phaser's physics engines) on Node.js.
</h4>

---

## Install

```console
npm install @geckos.io/phaser-on-nodejs
```

## How to use

```js
require('@geckos.io/phaser-on-nodejs')
// or with es6
import('@geckos.io/phaser-on-nodejs')
```
